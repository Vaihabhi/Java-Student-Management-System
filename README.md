# Java-Student-Management-System
Java-based application for managing student records (Under Development)
Under Development

This project is a Java-based application designed to manage student records such as name, course, marks, attendance, and performance data. The system allows adding, viewing, updating, and deleting student details.
It will be built using Core Java, OOPs, Collections, and file handling or a database.

Tech Stack

Java (Core + OOPs)

Collections Framework

File Handling / JDBC (planned)

IntelliJ / Eclipse
